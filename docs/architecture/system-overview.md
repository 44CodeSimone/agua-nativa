# System Overview

## Objective
Agua Nativa is a multi-tenant SaaS platform designed to manage rural water associations with professional standards, including customer management, units, meters, readings, billing, extra charges, payments, notifications, and reports.

## Core Applications
1. Central administrative platform
2. Association web panel
3. Meter reader mobile app
4. Customer mobile app

## Architecture Layers
- Presentation layer

- Infrastructure layer
